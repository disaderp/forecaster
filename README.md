# Forecaster

Physical weather forecast displayer AVR:[![Build Status](https://travis-ci.org/disaderp/forecaster.svg?branch=master)](https://travis-ci.org/disaderp/forecaster) ANDROID:![completion](https://img.shields.io/badge/completion-0%25-red.svg?maxAge=3600)

## what are the features of 'display'

Currently planned weather conditions are:
- sunny/cloudy
- rain
- lightning
- temperature display
- lunar phase
- ~~wind~~

## how it works

Display is controlled by AVR microcontroller([for more info](https://github.com/disaderp/forecaster/tree/master/SCHEM))
Forecast data for display will be transferred via BT from Android device
